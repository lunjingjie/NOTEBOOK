# 闭包
闭包是指有权访问另一个函数作用域中的变量的函数。

## 例子
```javascript
function createFunctions(){
    var result = new Array();

    for(var i=0; i < 10; i++){
        result[i] = function(){
            console.log(i);
        }
    }
     return result;
}

var result = createFunctions();

result[0](); // 10
result[1](); // 10
result[2](); //   
result[3](); // 10
result[4](); // 10
result[5](); // 10
```

结果输出都为10。

## 原理
函数内部没有i变量，函数会从作用域链上按顺序去寻找i变量。

## 解决方法
1.使用let、const代替var，使得变量不会被污染；  
2.使用立即函数封装；
